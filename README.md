# Visual SLAM Navigation Without GPS

This project demonstrates a Visual SLAM-based navigation system using a monocular RGB camera and ORB-SLAM3.  
The goal is to estimate camera movement, build a basic map of the environment, and support GPS-free navigation in indoor spaces.

## Main Features

- Monocular RGB camera input
- ORB-SLAM3 launcher script
- OpenCV-based Visual Odometry fallback
- Camera calibration configuration
- Trajectory visualization
- Documentation, report, and presentation slides

## Technologies

- Python
- OpenCV
- ORB-SLAM3
- C++
- Linux / Ubuntu
- Computer Vision
- Visual Odometry
- SLAM
